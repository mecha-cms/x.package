Package Extension for [Mecha](https://github.com/mecha-cms/mecha)
=================================================================

![Code Size](https://img.shields.io/github/languages/code-size/mecha-cms/x.package?color=%23444&style=for-the-badge)

This extension provides various API to work with [ZIP](http://www.php.net/manual/en/book.zip.php "PHP Â· ZIP") files. ZIP is an archive file format that supports lossless data compression. A ZIP file may contain one or more files or directories that may have been compressed. ZIP files generally use the file extensions `.zip` and the MIME media type `application/zip`.
